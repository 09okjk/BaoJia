# Retrieval Rules

## 目的

定义第三个 Skill 第一版的规则式检索口径。

## 基础检索维度

- 服务类别
- 服务模式
- 地点类型
- 可报价项标题
- 船型或其他全局特征

## 第一版建议策略

1. 先按服务类别和服务模式做硬过滤或高权重匹配
2. 再按 `quotable_items` 标题做关键词重合度匹配
3. 最后补充地点类型、船型等特征加权

## 相似度原则

- 相似度范围为 `0.0 - 1.0`
- 命中核心服务模式和核心项目标题时应明显提高分值
- 仅命中弱特征时应保持保守分值

## 置信度原则

- 有多个高相似样本时可提高 `confidence`
- 样本少或特征弱时应降低 `confidence`
- 不要因为存在单条模糊样本就给出高置信度
